# react-svg-polygon

### Install
``` bash
npm i react-svg-polygon
```

### Usage

``` jsx
    <Poly sides={3} r={30} strokeWidth={2} />
    <Poly
      sides={4}
      cx={50}
      cy={50}
      width={100}
      height={100}
      r={30}
      stroke={"#18efff"}
    />
    <Poly sides={5} r={30} fill={"red"} stroke={"none"} />
    <Poly sides={6} r={30} />
```
